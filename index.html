Install

  1. Step 1.The first step to installing the Scratch Agar.io Bot is to get a userscript manager. If you're using Firefox, Greasemonkey is a great option. If you're on Google Chrome, use Tampermonkey. There is also a version of Tampermonkey for Opera. These userscript managers are what allow us to install the script.
  2. Step 2.The second step requires that you click two links, and then click "allow", "confirm", or a similar option. This will add the bot scripts to your userscript manager. The links are here and here.
  3. Step 3.Now all that's left to do is to head over to agar.io and let the bot run! Simply visit the website and your game should start automatically. If the regular menu pops up and the game does not start, something went wrong.

    (Optional: If you'd like to change the agar.io settings at any time, just press the esc key on your keyboard.)

Contribute

The Scratch Agar.io Bot is made by the Scratch community. If you are a Scratcher who would like to help contribute, click the ribbon in the top right and create a branch of the repo. Once you've commited a few helpful changes, create a pull request and Firedrake969 will review what you've done. If your change is accepted, you will be added to the Scratch Agario Bots Community.
